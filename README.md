## 🛠️ Global Prerequisites

To run any of the projects inside this repository, you need to have **Ollama** installed on your machine and the base model downloaded:
```bash
# Install Ollama from [https://ollama.com](https://ollama.com)
# Then pull the core architecture model
ollama pull llama3.1:8b
```
